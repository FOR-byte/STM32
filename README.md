S型算法：

这里采用的是数学匀变速的数学模型，但是速度也并不是匀速变化的，所以准确的说是加速度的匀变数学模型。

首先有了以上的模型就可以算出每一时刻的加速度，所以先看 0~t/2 这段 a=K*T ，

其中K为加速度a的斜率将加速度积分就是速度，所以 V = a dt = K*T dt 得： V = K*T^2/2树

将速度积分就是位移，所以 S = V dt = K*T^2/2 dt 得： S = K*T^3/6
